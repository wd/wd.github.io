+++ title = "周末ktv的地址" tags = ["heart"] status = "publish" type = "post" published = true comments = true date = "2005-11-02" description = "" +++ 地点：佰金KTV，西直门外，交大东路25号。
路线：
西直门西侧桥下第一个红绿灯往北拐，上高梁桥斜街；
沿高梁桥斜往西北走，第一个红绿灯往北拐，上交大东路；
沿交大东路往北走，大概800米远处是一个没有红绿灯的十字路口；
十字路口的西北角，就是东方八喜大酒楼；
佰金KTV就在东方八喜大酒楼的南侧。
佰金KTV电话：82165558
