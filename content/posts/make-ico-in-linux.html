+++ title = "Linux下面如何制作ico图标？" tags = ["other"] status = "publish" type = "post" comments = true date = "2005-12-05" description = "" +++ 首先用画图工具制作png or gif文件，然后用netpbm转一下就可以了。
netpbm可以从这里下载：netpbm.sf.net
在debian下面，直接apt-get install netpbm即可。
然后这样转换：
pngtopnm myicon.png | ppmtowinicon -output favicon.ico
