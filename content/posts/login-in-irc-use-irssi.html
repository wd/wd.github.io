+++ title = "使用irssi上irc" tags = ["irssi", "other"] status = "publish" type = "post" comments = true date = "2006-09-09" description = "" +++ /SERVER ADD -auto -network oftc irc.oftc.net 6667
/NETWORK ADD -autosendcmd "/^msg nickserv identify pass;wait 2000" OFTC
/CHANNEL ADD -auto #arch-cn OFTC
这样可以实现起动irc自动登录，并且加入频道
irssi 的 manual 以及一个Howto 。
