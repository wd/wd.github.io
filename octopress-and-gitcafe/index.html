octopress and gitcafe

ocotpress 支持 github，不过 github 的 pages 貌似被墙了，且速度慢。gitcafe 速度还不错，也支持 pages，刚好可以切换过去。我只是简单粗暴的修改了几个文件。

+desc "deploy gitcafe"
+task :deploy do

+  puts "\n## copying #{public_dir} to #{gitcafe_dir}"
+  cp_r "#{public_dir}/.", gitcafe_dir
+  cd "#{gitcafe_dir}" do
+    system "git add ."
+    system "git add -u"
