qss不正常显示的问题

2006/01/16

Tags: Mail

1 php3.x版本以上不正常显示
设置php.ini的register_globals = On
或者
修改getGraph.php添加
$data=$_GET['data'];
$t=$_GET['t'];
修改getGraph1.php添加
$data=$_GET['data'];
$s=$_GET['s'];
2 还不能正常显示
那可能你的qmail-scanner的版本是1.2以上的，此版本以上的log文件格式有些变化，qss却没有做相应更新，所以会有问题。你可以看看你的log文件的日期是不是下面这种格式
Mon, 03 Jan 2005 04:01:57 CST
如果是，那么差不多就是这个问题了。
解决方法是在index.php里面查找
# Calc the date timestamp
把他下面的5行语句用下面的替换
if ( eregi("(^[0-9]+)", $val[0]) )
{
# Calc the date timestamp
$date = explode("/",$val[0]);
$dateT = $date[0];
$date[0] = $date[1];
$date[1] = $dateT;
$date = strtotime(implode("/",$date));
}
else
{
$date = strtotime ($val[0]);
};
